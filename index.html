Demon Slayer Hinokami Chronicles 2 Modding Guide Part 1

A comprehensive guide to modding Demon Slayer Hinokami Chronicles 2, covering file extraction, UE4 modding basics, and texture replacement.

By jpthebaby Last updated: June 2023 Reading time: 15 min UE4, Modding, Demon Slayer

Introduction

This tutorial will guide you through the process of modding Demon Slayer Hinokami Chronicles 2. The game runs on Unreal Engine 4, which makes it relatively straightforward to mod once you understand the basics.

Note: This guide assumes you have basic computer skills and are comfortable with file management. Always back up your game files before making any changes.

We'll be covering the entire process from extracting game files to implementing your mods. The tutorial is broken down into several parts to make it easier to follow.

Requirements

Before you begin, make sure you have the following tools and software installed:

1
Essential Tools
  • FModel: For extracting UE4 game files
  • UE4.27 or UE4.25: Unreal Engine version
  • UModel: Alternative tool for asset extraction
  • 7-Zip or WinRAR: For archive management
2
Optional Tools
  • Photoshop or GIMP: For texture editing
  • Blender: For 3D model editing
  • Notepad++: For editing configuration files
Warning: Modding online games may violate terms of service. Only mod in offline/single-player modes unless you understand the risks.

File Extraction

The first step in modding any UE4 game is to locate and extract the game files. These are usually stored in .pak files within the game directory.

Locating Game Files

Navigate to your game installation directory. For Steam games, this is typically:

# Steam installation path
C:\Program Files (x86)\Steam\steamapps\common\Demon Slayer Hinokami Chronicles 2

Using FModel for Extraction

  1. Open FModel and set the game path to your installation directory
  2. Select the appropriate UE4 version (4.27 or 4.25)
  3. Navigate to the .pak files you want to extract
  4. Right-click and select "Extract" to save files to your desired location

Most game assets will be in the "Content" folder. Textures are usually in .uasset or .utexture files, while models are in .umap or .uasset files.

Modding Process

Once you've extracted the necessary files, you can begin the modding process. This typically involves:

1
Asset Replacement

Replace existing game assets with your custom ones. This is the most common type of mod, especially for texture replacements.

2
Script Modification

Edit game scripts to change behavior, add new features, or fix bugs. This requires more advanced knowledge of UE4 and programming.

3
Packaging Mods

Once your mod is ready, you'll need to package it properly so it can be installed by other users. This usually involves creating a .pak file with your modified assets.

UE4 Modding Basics

Understanding UE4's file structure is crucial for successful modding. Here are some key concepts:

UE4 File Types

  • .uasset: Contains game assets like textures, materials, and blueprints
  • .umap: Contains level/map data
  • .uexp: Binary data associated with .uasset files
  • .ubulk: Bulk data like high-resolution textures
  • .pak: Archive containing multiple game files

Mod Installation

Most UE4 mods are installed by placing .pak files in the game's "Paks" folder. Some mods may require additional steps like creating a "~mods" folder.

# Typical mod installation path
GameDirectory\Content\Paks\~mods\

Texture Modding

Texture modding is one of the most accessible forms of modding. Here's a basic workflow:

  1. Extract the texture you want to modify using FModel
  2. Convert the texture to a standard format like .png or .dds
  3. Edit the texture in your preferred image editor
  4. Convert back to the game's format
  5. Repackage and test in-game
Pro Tip: When editing textures, always maintain the same dimensions and format as the original to ensure compatibility.

Conclusion

This concludes Part 1 of the Demon Slayer Hinokami Chronicles 2 modding guide. You should now have a basic understanding of:

  • Required tools for UE4 modding
  • How to extract game files
  • The basics of the UE4 file structure
  • An overview of the modding process
